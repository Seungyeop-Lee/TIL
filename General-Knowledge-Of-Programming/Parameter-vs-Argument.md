# Parameter vs Argument
- Parameter(매개변수)
  - 메소드 안에서의 매개변수
- Argument(인수)
  - 메소드를 호출 할 때의 인수
```JavaScript
function sum(a, b) {  //a, b는 Parameter
  return a + b;
}

int resultSum = sum(10, 20);  //10, 20은 Argument
```
